# Relativity Simulator 🌌

An interactive and pedagogical General Relativity physics simulator built with React and 3D web technologies.

## Features

- **Interactive 3D Simulations**: Explore black holes, gravitational lensing, and spacetime curvature using Three.js and React Three Fiber.
- **Pedagogical Content**: Educational modules explaining the core concepts of General Relativity with beautiful mathematical formulas rendered via KaTeX.
- **Modern UI**: Designed with TailwindCSS and Framer Motion for smooth animations and a premium, responsive user experience.
- **Client-Side Rendering**: Fast and lightweight, running entirely in the browser using Vite.

## Technologies Used

- [React 18](https://reactjs.org/)
- [Vite](https://vitejs.dev/)
- [Three.js](https://threejs.org/) & [@react-three/fiber](https://docs.pmnd.rs/react-three-fiber/getting-started/introduction)
- [Tailwind CSS v4](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)
- [KaTeX](https://katex.org/)

## Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone or extract this repository.
2. Open a terminal in the project directory.
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## Scripts

- `npm run dev`: Starts the development server.
- `npm run build`: Builds the application for production.
- `npm run preview`: Previews the production build locally.
- `npm run lint`: Runs ESLint for code formatting checks.
